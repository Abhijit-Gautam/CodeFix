## Solutions

### EASY - Wumpus World Navigator (A* Pathfinding)
- **File**: `EASY/wumpus_world.py`
- **Description**: Implemented A* pathfinding algorithm for navigating a Bangalore-themed Wumpus World grid
- **Key Implementation**:
  - `get_cell_cost()` - Returns appropriate costs for each cell type
  - `heuristic()` - Manhattan distance implementation
  - `find_path_astar()` - Complete A* algorithm with priority queue

### MEDIUM1 - Database Insights Agent
- **File**: `MEDIUM1/db_analyzer.py`
- **Description**: Intelligent agent that analyzes SQLite databases and generates comprehensive reports
- **Usage**: `python db_analyzer.py --db data.db --email recipient@example.com`
- **Features**:
  - Auto-discovers database schema
  - Generates statistics and insights
  - Creates 3+ visualizations
  - Generates HTML and PDF reports
  - Email automation with attachments

### MEDIUM2 - Fashion MNIST Classifier
- **File**: `MEDIUM2/fashion_classifier.py`
- **Description**: Image classifier using pre-trained Keras model for Fashion MNIST dataset
- **Usage**: `python fashion_classifier.py --model model.h5 --directory images/`
- **Features**:
  - Loads pre-trained model
  - Preprocesses images (grayscale, resize, normalize)
  - Classifies single images or directories
  - Calculates accuracy against true labels
